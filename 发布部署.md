### 使用PM2启动Node服务

背景：如果使用传统方式启动服务，命令行会被占用，一旦退出命令行服务就被终止。为解决此问题，我们可以使用PM2进程管理工具。

概念：PM2是专为管理nodejs进程的一款工具，可以利用它来简化很多node应用管理的繁琐任务，如性能监控、自动重启、负载均衡等，而且使用非常简单。

官方文档：https://pm2.io/

安装：npm install pm2 -g

启动：pm2 start 脚本路径
